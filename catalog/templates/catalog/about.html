About us:

Collectio is a catalog collection system that will allow it's users to keep track of the objects in their collections. This webpage provides a system that is suitable for individual collectors and small museums. It is easy to use and very flexible in nature. The live version of the system is to be run locally since production is still in Alpha version. We plan on adding more features to it with time.
